Numeric Question Type: The "digits" parameters

Use Max. digits allowed to set a limit to the number of characters entered for a Numeric question. Note that the decimal point also counts as one character!

Use Nb of decimal digits to specify the format of the Average value counted and displayed at the Questionnaire Report page.
